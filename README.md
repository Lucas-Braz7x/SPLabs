# SPLabs

Desafio proposto pela empresa "Sem processo"

## Como rodar:

  - Baixe ou faça um clone do repositório `git clone urlRepositorio`
  - Instale as dependências `npm install`
  - Inicie o projeto `npm start`

## Tecnologias:

  - React
  - Redux
  - Sass
  - Axios
